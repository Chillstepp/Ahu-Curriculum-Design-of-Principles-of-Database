# 使用方法

## 1 配置python环境
建议使用anaconda+pycharm的组合

[Anaconda](https://www.anaconda.com/)

[Pycharm](https://www.jetbrains.com/pycharm/download/)

## 2 下载django
命令行输入 `conda install django` / `pip install django`

## 3 运行
cd 至根目录下 `python manage.py runserver`
